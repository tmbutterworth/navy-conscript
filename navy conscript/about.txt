A plugin to join The Republic Navy as a conscript.

This Plug-in provides an alternate story plot to the main story plot. You can not be both a Navy Conscript and a Free Worlds Militia member. 

This plug-in uses time based events if you game is past: 11 7 3014 the plug-in story plot will never start. You will either need to start a new game or modify the date in your save file to be before 11 7 3014.
